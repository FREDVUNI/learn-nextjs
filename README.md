Explore these resources to get started with Next.js:

- **Description**: The official documentation provides comprehensive information on Next.js, including its core concepts, features, and how to get started with building applications.

- **Description**: Explore official examples to see practical implementations of Next.js features, making it easier to learn through real-world code.
